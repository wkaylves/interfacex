## 🚀 Quick Start

InterfaceX helps you quickly locate all interface entry points in large projects:

- **🌐 HTTP Endpoints** - Spring-MVC controllers
- **🔄 OpenFeign Calls** - Remote service dependencies
- **📮 RabbitMQ** - Producers and consumers
- **🚀 RocketMQ** - Message queue management
- **⏰ XXL-JOB** - Scheduled task execution and debugging
- **🏷️ Tag Management** - Professional interface classification
- **🔗 Line Markers** - Smart code navigation

👉 **For detailed features, installation guide, and usage instructions, please visit our [Official Website](https://wkaylves.github.io/interfacex/)**

## 📦 Installation

### From JetBrains Marketplace (Recommended)

1. Open IntelliJ IDEA
2. Go to `Settings/Preferences` → `Plugins`
3. Search for "InterfaceX"
4. Click `Install`
5. Restart IDE

### Manual Installation

Download the latest release from [GitHub Releases](https://github.com/wkaylves/interfacex/releases) and install manually.
