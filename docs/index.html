✨ 核心特性

InterfaceX 是您在大型项目中的得力助手，自动扫描并清晰展示各类组件入口点

🌐

HTTP 接口

快速定位 Spring-MVC 控制器，支持 @GetMapping、@PostMapping 等注解

🔄

OpenFeign 调用

清晰展示远程服务间的调用关系，轻松掌握微服务依赖

📮

RabbitMQ

可视化消息的发送与消费逻辑，自动识别 RabbitTemplate 和监听器

🚀

RocketMQ

支持阿里云生态的消息队列管理，完整追踪消息流向

⏰

XXL-JOB 任务

直接在 IDE 内执行和调试定时任务，告别繁琐的容器环境配置

🏷️

标签管理系统

专业的接口分类和管理能力，支持搜索、过滤、批量操作

🔗

智能行标记导航

在代码编辑器侧边栏提供智能导航图标，实现组件间快速跳转

💾

SQLite 持久化

高性能数据库存储，支持动态切换和自动迁移，大幅提升加载速度

🎯 解决的核心痛点

场景 传统方式的困扰 InterfaceX 如何解决
大型项目 代码量庞大，全局搜索效率低下 自动分类扫描，一键展示所有接口入口
微服务调用 难以追溯调用链路 清晰列出所有 OpenFeign 调用方
消息队列调试 需要翻阅大量代码 自动识别生产者和消费者，可视化呈现
XXL-JOB 调试 本地断点无法生效 IDE 内直接触发任务，本地调试畅通无阻

📦 安装指南

1

从 JetBrains 插件市场安装（推荐）

  1. 打开 IntelliJ IDEA
  2. 进入 Settings/Preferences → Plugins
  3. 搜索 "InterfaceX"
  4. 点击 Install 按钮
  5. 重启 IDE
2

手动安装

  1. 从 GitHub Releases 下载最新版本的 .zip 文件
  2. 打开 IntelliJ IDEA
  3. 进入 Settings/Preferences → Plugins
  4. 点击齿轮图标，选择 Install Plugin from Disk...
  5. 选择下载的 .zip 文件
  6. 重启 IDE

📖 使用说明

打开 InterfaceX 工具窗口

安装后，您可以在以下位置找到 InterfaceX：

  • 右侧工具栏中的 "InterfaceX" 标签
  • 菜单栏：View → Tool Windows → InterfaceX

刷新接口列表

点击工具窗口顶部的刷新按钮，插件将自动扫描当前项目中的所有接口。

使用标签管理

右键点击任意接口，可以：

  • 添加/移除标签
  • 按标签过滤视图
  • 打开专业标签管理器

执行 XXL-JOB 任务

右键点击 XXL-JOB 任务节点，选择"执行任务"，设置端口号后即可在本地调试。
