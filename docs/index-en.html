📦 Installation Guide

1

Install from JetBrains Marketplace (Recommended)

  1. Open IntelliJ IDEA
  2. Go to Settings/Preferences → Plugins
  3. Search for "InterfaceX"
  4. Click Install button
  5. Restart IDE
2

Manual Installation

  1. Download the latest .zip file from GitHub Releases
  2. Open IntelliJ IDEA
  3. Go to Settings/Preferences → Plugins
  4. Click gear icon, select Install Plugin from Disk...
  5. Select the downloaded .zip file
  6. Restart IDE

📖 Usage Instructions

Open InterfaceX Tool Window

After installation, you can find InterfaceX at:

  • "InterfaceX" tab in the right toolbar
  • Menu bar: View → Tool Windows → InterfaceX

Refresh Interface List

Click the refresh button at the top of the tool window, the plugin will automatically scan all interfaces in the current project.

Use Tag Management

Right-click on any interface to:

  • Add/remove tags
  • Filter view by tag
  • Open professional tag manager

Execute XXL-JOB Tasks

Right-click on XXL-JOB task node, select "Execute Task", set port number to debug locally.
